# Log Analyzer for Security Monitoring

A Python-based tool to analyze system logs and detect suspicious activities such as failed login attempts.

## Features
- Parses system logs
- Detects failed login attempts
- Identifies suspicious IP patterns

## Technologies Used

- Regex
